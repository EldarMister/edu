# Требования к типографике интерфейса

Типографика должна строго соответствовать подготовленным референсам из папки:

`/designe`

Интерфейс не должен выглядеть тяжёлым. Нельзя делать все тексты слишком жирными, крупными или контрастными без необходимости.

## Общий стиль текста

Текст должен быть:

* аккуратным;
* спокойным;
* читаемым;
* современным;
* не слишком жирным;
* без лишнего визуального шума.

Основной стиль интерфейса — лёгкий, чистый, как в референсах.

---

## Рекомендуемый шрифт

Для интерфейса использовать системный современный sans-serif шрифт:

```css
font-family: Inter, system-ui, -apple-system, BlinkMacSystemFont, "Segoe UI", sans-serif;
```

Если Inter не подключается, использовать системный шрифт.

---

## Вес шрифтов

Использовать ограниченное количество font-weight:

### Заголовки страниц

Например:

* Кухня
* Авторизация
* Заказы
* Меню
* Администрация

Параметры:

```css
font-size: 28px - 32px;
font-weight: 600;
line-height: 1.2;
```

Не использовать `font-weight: 800` или `900`.

---

### Заголовки карточек

Например:

* №000125
* Заказ №000125
* Вход в систему
* Название блюда

Параметры:

```css
font-size: 18px - 22px;
font-weight: 600;
line-height: 1.3;
```

---

### Основной текст

Например:

* Официант: Иванов И.
* Стол 5
* Время заказа
* Названия блюд
* Комментарии

Параметры:

```css
font-size: 14px - 16px;
font-weight: 400;
line-height: 1.4;
```

---

### Важные значения

Например:

* сумма заказа;
* количество;
* таймер ожидания;
* номер стола;
* итоговая сумма.

Параметры:

```css
font-size: 16px - 20px;
font-weight: 500 или 600;
line-height: 1.3;
```

Использовать жирность только там, где это действительно важно.

---

### Второстепенный текст

Например:

* комментарий к блюду;
* подписи;
* время;
* статус;
* описание;
* вспомогательная информация.

Параметры:

```css
font-size: 12px - 14px;
font-weight: 400;
color: #64748B;
line-height: 1.4;
```

---

## Кнопки

Текст в кнопках должен быть читаемым, но не чрезмерно жирным.

Параметры:

```css
font-size: 15px - 16px;
font-weight: 500;
```

Для главных кнопок можно использовать:

```css
font-weight: 600;
```

Но нельзя делать все кнопки слишком жирными.

---

## Что запрещено

Запрещается:

* делать весь текст `font-weight: 700`, `800` или `900`;
* делать слишком большие заголовки без причины;
* использовать много разных размеров текста;
* использовать много разных шрифтов;
* делать интерфейс похожим на рекламный баннер;
* делать слишком тёмный и тяжёлый текст везде;
* добавлять декоративные шрифты;
* делать текст плотным и перегруженным;
* использовать жирный шрифт для всех названий блюд, комментариев и статусов.

---

## Цвет текста

Основные цвета текста:

```css
--text-primary: #0F172A;
--text-secondary: #475569;
--text-muted: #64748B;
--text-light: #94A3B8;
--primary-blue: #005BFF;
--danger-red: #EF4444;
--success-green: #16A34A;
--border-color: #E2E8F0;
--background: #F8FAFC;
--card-background: #FFFFFF;
```

Основной текст должен быть тёмным, но не слишком тяжёлым.

---

## Принцип типографики

В интерфейсе должно быть максимум 4 уровня текста:

1. Заголовок страницы
2. Заголовок карточки или блока
3. Основной текст
4. Второстепенный текст

Не нужно создавать много разных размеров и жирностей. Чем меньше вариантов текста, тем чище будет интерфейс.

---

## Обязательное условие

Перед вёрсткой frontend-разработчик обязан открыть референсы из папки `/designe` и повторить типографику по ним:

* размер текста;
* жирность;
* расстояния между строками;
* отступы;
* лёгкий визуальный стиль.

Если разработчик сомневается, лучше сделать текст чуть легче, чем слишком жирным.
